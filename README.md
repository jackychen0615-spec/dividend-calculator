# 股利計算器

免費線上投資計算工具平台，提供多種台股投資試算工具。

🔗 **網站：** https://dividend-calculator-eym.pages.dev

## 功能

### 計算工具
- **股利計算器** — 輸入股價、股數與每股股利，算出投入成本、總股利與殖利率
- **存股複利計算器** — 計算長期存股的複利成長效果
- **存股回本計算器** — 算出靠股利回本需要幾年
- **定期定額試算器** — 每月定期投入的長期投資成果

- 台股配息查詢完整指南
