勞保、勞退、國民年金差在哪？2026完整比較懶人包

✎ Jacky Chen 2026-05-29

「勞保」、「勞退」、「國民年金」這三個名詞常常被搞混，很多人以為勞保就是勞退，或是不知道國民年金跟自己有什麼關係。其實這三者是完全不同的制度，保障範圍、繳費方式、領取條件都不一樣。這篇文章用一張比較表加上實際試算，幫你一次搞懂。

簡單來說：勞保是「保險」，有各種給付保障；勞退是「存錢」，雇主幫你存退休金；國民年金是「沒工作時的基本保障」。

勞工保險（勞保）詳解

勞保是台灣最重要的勞工社會保險，涵蓋五大給付：

  • 生育給付：女性被保險人分娩，可領2個月投保薪資。
  • 傷病給付：因傷病住院不能工作，第4天起可領投保薪資50%（普通傷病）或70%（職業傷病）。
  • 失能給付：因傷病致身體失能，依失能等級領取一次金或年金。
  • 老年給付：達到法定年齡且符合年資條件，可領老年年金或一次金。
  • 死亡給付：被保險人死亡，遺屬可領喪葬津貼及遺屬年金。

勞保老年年金怎麼算？

老年年金有兩個計算公式，擇優發給：

公式一：平均月投保薪資 x 年資 x 0.775% + 3,000元

公式二：平均月投保薪資 x 年資 x 1.55%

以平均月投保薪資40,000元、年資30年為例：

  • 公式一：40,000 x 30 x 0.775% + 3,000 = 12,300元/月
  • 公式二：40,000 x 30 x 1.55% = 18,600元/月

擇優選公式二，每月可領18,600元。

請領年齡規定

勞保老年年金的法定請領年齡逐年調升：

年度 法定請領年齡
2024年63歲
2026年64歲
2028年及以後65歲

你也可以提前最多5年請領（每提前1年減給4%，最多減20%），或延後最多5年（每延後1年增給4%，最多增20%）。

勞工退休金（勞退新制）詳解

勞退新制從2005年7月開始實施，採「個人帳戶制」，每位勞工都有一個專屬的退休金帳戶。

雇主提撥

雇主每月必須提撥勞工月提繳工資的6%到勞工的個人帳戶。這筆錢是雇主的額外成本，不會從勞工薪水扣除。帳戶內的錢由勞保局統一運用投資，設有最低保證收益（不低於2年期定存利率）。

勞退自提

勞工可以自願提撥0%到6%的月提繳工資到同一帳戶。自提的好處是自提金額可以從當年度個人綜合所得總額中全數扣除，達到節稅效果。詳細分析請參考勞退自提6%節稅效果試算。

勞退可以領多少？試算範例

假設月薪40,000元，工作30年，雇主每月提撥6%（2,400元），不考慮薪資調整：

項目 僅雇主提撥 雇主提撥+自提6%
每月提撥金額2,400元4,800元
30年累計本金864,000元1,728,000元
加計收益（年化2%估算）約1,180,000元約2,360,000元
月領金額（分24年領）約4,100元/月約8,200元/月

可以看到，如果有自提6%，退休金帳戶的累積金額會翻倍。搭配勞保老年年金，退休後每月可領的金額就更可觀。

國民年金詳解

國民年金是針對25歲以上、未滿65歲、沒有參加勞保或其他社會保險的國民所設計。常見的加保對象包括：

  • 家庭主婦或主夫
  • 待業中、找工作的人
  • 未加入職業工會的自由工作者
  • 替代役男

2026年國民年金費率為10.5%，月投保金額為19,761元，個人負擔60%，每月保費約為1,245元。65歲之後可按月領取老年年金，基本計算為：

月投保金額 x 保險年資 x 0.65% + 3,772元

或

月投保金額 x 保險年資 x 1.3%

擇優計算。繳滿40年的話，每月最多可領約10,000元上下。金額雖然不高，但對於沒有其他退休保障的人來說，是基本的經濟安全網。

三者可以同時領嗎？

可以，但有條件：

  • 勞保老年年金 + 勞退：退休時可以同時領取，兩者完全獨立。
  • 勞保 + 國民年金：如果你曾經有勞保年資，也有國民年金年資，退休後兩者都可以請領，但國民年金的計算公式會受限（不能使用有加計3,772元的公式）。
  • 勞退 + 國民年金：可以同時領取，互不影響。

給不同族群的建議

上班族

你同時有勞保和勞退保障。建議確認雇主是否有按實際薪資投保（高薪低報是違法的），並考慮是否要勞退自提。可以用薪資計算器查看每月的扣款明細。

自由工作者

如果有加入職業工會，可以透過工會參加勞保。沒有加入任何社會保險的話，會自動被納入國民年金。建議至少維持國民年金的繳費，確保老年有基本保障。

即將退休者

提前5年開始規劃：確認勞保年資是否足夠、勞退帳戶累積了多少錢、要選月領還是一次領。可以到勞保局網站試算，或使用薪資計算器先了解目前的提撥狀況。

常見問題 FAQ

勞保和勞退有什麼差別？

勞保是社會保險，提供生育、傷病、失能、老年、死亡五大給付，採用共同基金制。勞退是個人退休金帳戶，雇主每月提撥薪資6%到你的專屬帳戶，退休後領回本金加收益。兩者是獨立制度，退休時可以同時領。

勞保年金幾歲可以開始領？

2026年勞保老年年金的法定請領年齡為64歲。你也可以提前最多5年請領（每提前1年減給4%），或延後最多5年請領（每延後1年增給4%，最多增加20%）。2028年起法定年齡調升至65歲。

國民年金是什麼？誰要繳？

國民年金是針對25歲以上、未滿65歲、沒有參加勞保或其他社會保險的國民所設計的基礎保障。常見繳納族群包括家庭主婦/主夫、待業中的人、未加入職業工會的自由工作者。每月保費約1,245元。

勞退的錢可以提前領出來嗎？

原則上不行。勞退新制的個人帳戶需年滿60歲才能請領。工作年資滿15年以上可選擇月領或一次領，未滿15年則只能一次領。唯一例外是勞工身心障礙經評估不堪工作，可提前請領。
