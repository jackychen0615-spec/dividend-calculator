常見問題

股利的計算方式很簡單：總股利 = 持有股數 x 每股股利。例如你持有 1,000 股（1 張）某檔股票，該股票每股配發 2 元股利，那麼你可以領到 1,000 x 2 = 2,000 元的股利。
殖利率（Dividend Yield）是衡量股票配息報酬率的指標，公式為：殖利率 = 每股股利 / 股價 x 100%。例如股價 50 元、每股配 2 元股利，殖利率就是 2 / 50 x 100% = 4%。一般來說，殖利率超過 5% 就算高殖利率股票。
一張股票等於 1,000 股。所以一張股票的股利 = 1,000 x 每股股利。例如某檔股票每股配 3 元現金股利，持有一張就能拿到 3,000 元。你可以使用上方的計算器輸入股數與每股股利，立即算出結果。
現金股利是公司直接發放現金到你的帳戶；股票股利則是公司發放額外的股票給你。例如配發 0.5 元股票股利，代表每持有 1,000 股可以多拿 50 股。多數存股族偏好穩定的現金股利，因為可以直接當作被動收入使用。
